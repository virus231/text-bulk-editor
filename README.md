# Text Bulk Editor

## Як запустити локально
Передумови: встановлений Bun.

```bash
bun install
bun run dev
```

Додатково:
- `bun run build` — типізація + продакшн-збірка.
- `bun run preview` — локальний перегляд продакшн-білду.

## Рішення щодо продуктивності / undo-redo
- **Асинхронні операції над текстом.** Частина трансформацій може працювати в кілька кадрів (Promise), тому в `useTextEditor` обробка запускається через `setTimeout` і вимірюється тривалість виконання.
- **Захист undo/redo під час виконання.** Поки `isWorking === true`, `undo` та `redo` ігноруються, а кнопки/шорткати фактично вимкнені (`canUndo`/`canRedo` повертають `false`). Це запобігає перезапису історії, коли довга операція ще не завершена.
- **Обмеження історії.** Зберігається до 20 кроків (`MAX_HISTORY`) — баланс між зручністю та пам’яттю.

## Що не встигли / що б покращили
- Додати показ прогресу для довгих операцій і кнопку “Скасувати”.
- Покрити критичну логіку (undo/redo, трансформації) тестами.
- Зробити імпорт/експорт із більшими форматами (CSV) та опціями кодування.
